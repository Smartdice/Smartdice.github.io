About Us

TTOVI is a Smartdice project to create an application that helps its user track the time spent working on tasks in a non-invasive manner. We aim to implement an innovative solution and optimize the process in comparison to already existing time tracking applications. The tasks are created a priori and are to be downloaded from a project management tool (PMT), and the times registered need to be uploaded to the same platform.
