About

Dali is a Deloitte Digital project to create an innovative way to provide immersive experiences to their customers. It aims to make it possible to analyze and explore data in innovative and unconventional ways and taking advantage of virtual reality technologies. Data analysis can be very dull and boring and the main goal of this project is to minimize these factors.

Product

Dali offers an innovative way to analyze data while breaking the existent preconceptions that we have when thinking of doing these tasks. Instead of presenting the user with many boring spreadsheets or a complicated graphs, Dali represents data pieces as 3D spheres that the user can interact with in order to know more about what they're exploring. In our banking example, different color shades represent total revenue a single manager brought to the bank. The size of each sphere represents the number of clients they are assigned to. Spheres can also be filtered using the buttons on the table. Interacting with a sphere displays a popup with more details about that manager. All of these interactions are triggered by the user's hand movements.
